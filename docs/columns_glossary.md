# Media Plan — Column Glossary

| Column Header     | Definition                                                                                                                                                                                                                                                                             |
| ----------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Markets**       | The geographic region or city cluster being targeted in the campaign. Each market represents a distinct audience location (e.g. Maharashtra, Andhra Pradesh).                                                                                                                          |
| **Channel**       | The advertising platform and campaign phase. Indicates the platform (e.g. Meta, YouTube, WhatsApp) along with whether the activity is Pre-launch or During the campaign.                                                                                                               |
| **WOA**           | Weeks of Activity. The number of weeks a particular line item will be active and running ads.                                                                                                                                                                                          |
| **Targeting**     | The demographic segment being targeted for that line item. Includes age range and gender (M = Male, F = Female, M+F = All genders).                                                                                                                                                    |
| **Buy Type**      | The method by which ad inventory is purchased. 'RNF' (Reach & Frequency) is a Meta buying method that gives predictable reach and controlled frequency. 'Auction - TF' refers to a TrueView/auction-based buying method on YouTube. 'Fixed' refers to a reserved/fixed-cost placement. |
| **Asset**         | The creative format used for the ad. Describes the type of media delivered to the audience (e.g. Image/Video, 6 sec Video).                                                                                                                                                            |
| **Inventory**     | The specific ad placement or format location within a channel where the ad will appear (e.g. Feeds, Stories, Reels on Meta; Bumper Ads on YouTube; WhatsApp Message on WhatsApp).                                                                                                      |
| **Total Reach**   | The estimated total number of unique individuals who will see the ad at least once during the campaign period.                                                                                                                                                                         |
| **Avg Frequency** | The average number of times each unique reached user is expected to see the ad over the campaign duration.                                                                                                                                                                             |
